圖案編輯工具¶

在 4.0 版本開始棄用: 這個工具在 4.0 版本時已被移除，現在並沒有方法編輯向量的圖案填滿。

圖案編輯工具能對向量圖形中的填充圖案進行編輯。在這些形狀中，這個工具能改變圖案的尺寸、比例和原點。

在畫布中編輯¶

你可以透過拖曳上面的節點來更改原點，這只能在平鋪模式中使用。

你可以透過拖曳下方的節點來更改尺寸和比例。這裡沒有方法在畫布編輯中約束比例，這只能在原始和平鋪模式中使用。

工具選項¶

這裡有數項對於工具的選項，能作完全調整：

首先有圖案選項。

重覆：

這個可以設定為：

原始：

這只會顯示一個、未伸展，複製的圖案。

平鋪（預設）：

這會使圖案以 X 軸和 Y 軸平鋪。

伸展：

這會拉伸圖案影像以符合形狀。

參考點：

圖案的原始點。這個可以設定為：

  • 左上角

  • 頂部

  • 右上角

  • 左側

  • 中心

  • 右側

  • 左下角

  • 底部

  • 右下角

參考點偏移：

提供額外調整，以比率設定。

X:

以 X 軸偏移，即水平偏移。

Y:

以 Y 軸偏移，即垂直偏移。

平鋪偏移：

若是圖案為平鋪，則偏移圖案。

圖案大小：

精確調整圖案的大小。

W:

寬度，以像素為數值。

H:

高度，以像素為數值。

在這裡會有 圖案，即小型的圖案工具面板，你可以在這裡選擇填充的圖案。
