文字工具¶

這個工具可以在你的作品裡加上文字。

透過 + 拖曳 來新增一個矩形選取區域。當放開 時基本的文字段落會出現，同時彈出文字編輯界面。

游標移到其他文字方塊時會標示其邊框。 會把文字方塊設定為使用中的文字。

工具選項¶

建立新文字自...

這裡包含了建立新文字的功能，具有以下選項：

字型

設定建立文字時所使用的字型。

文字尺寸（pt）

設定建立文字時文字的尺寸。數值為 pts（點），這是 1 英寸比 72 點的標準尺寸。因此即使改變畫布的 dpi 時文字也會保持相同的尺寸。

錨定 / 對齊 文字至左側 / 中間 / 右側

文字對齊。可以對齊文字至左側、中間或是右側。稱為文字錨定的原因是 SVG 1.1 的多行文字只會使用文字錨定，這跟文字對齊有所分別（這也是沒有對齊功能的原因）。

字距

設定建立文字時字詞之間的間距。

編輯文字

為了使用中的物件，Krita 會彈出一個文字編輯器。可以透過 Enter 鍵，或是 點兩下 來快速取用。

文字編輯器¶

包含所有文字編輯需求的視窗。裡面包含兩個分頁：富文字和 SVG 原始碼。

啟用

使用文字工具來建立一個文字方塊。工具選項會有數項選項來改變建立文字時的樣式。在畫布裡拖曳一個矩形來建立文字區域。文字建立後，可以透過兩個方法來編輯文字：

  1. 透過形狀選取區域工具（第一個工具）來選取文字。按 Enter 鍵後文字編輯器便會出現。

  2. 透過形狀選取區域工具（第一個工具）來選取文字，然後切換到文字工具，工具選項裡有 編輯文字 按鈕，點擊按鈕後文字編輯器便會出現。

編輯

如果你不熟悉 SVG 文字的使用方式，請使用富文字分頁，這分頁可以用肉眼的方式來編輯文字，只是這個分頁沒有包含所有的功能。

如果你對 SVG 文字有一定的認知，你可以直接使用 SVG 原始碼來編輯文字。請注意不是所有屬性都能轉換回富文字編輯器，所以當要換為富文字編輯器時請留意。

當完成編輯後按 儲存 來在畫布中預覽文字。

檔案
儲存 Ctrl + S

儲存目前編輯的文字至畫布。

關閉 Ctrl + W

關閉編輯器。

編輯

基本編輯功能：

返回上一步 Ctrl + Z

返回文字編輯器裡的上一個步驟。

重做操作 Ctrl + Shift + Z

重做文字編輯器裡上一個未完成的操作。

剪下 Ctrl + X

剪下選取中的文字至剪貼簿。

複製 Ctrl + C

複製選取中的文字至剪貼簿。

貼上 Ctrl + V

貼上來自剪貼簿的文字。

選取所有 Ctrl + A

選取使用中的文字編輯器裡的所有文字。

取消選取 Ctrl + Shift + A

取消選取所有已選取的文字。

尋找 Ctrl +F

彈出一個視窗，輸入文字後會在文字編輯器裡尋找與輸入文字相符的文字。

尋找下一個 F3

尋找符合尋找文字的下一個文字。

尋找上一個 Shift + F3

尋找符合尋找文字的上一個文字。

取代... Ctrl + R

彈出有兩個輸入項目的界面：一個是要尋找的文字，另一個是想取代的文字。會取代所有符合的項目。

檢視
縮小 Ctrl + -

縮小文字。

放大 Ctrl + +

放大文字。

插入
特殊字元... Alt + Shift + C

彈出一個用來尋找難以用鍵盤輸入的特殊字元的界面。

格式
粗體 Ctrl + B

設定文字粗細為粗體。

斜體 Ctrl + I

設定選取文字為斜體。

底線 Ctrl + U

為選取文字添加底線。

刪除線

為選取文字添加刪除線。

上標 Ctrl + Shift + P

設定文字為上標。

下標 Ctrl + Shift + B

設定文字為下標。

粗細

設定更明確的文字粗細。有可能的選項為... 幼，普通，較粗，粗。

左側對齊

對齊選取段落至左側。

居中對齊 Ctrl + Alt + C

居中所選取的段落。

右側對齊 Ctrl + Alt + R

對齊選取段落至右側。

字距調整（Kerning）

調整選取文字的字距。

設定
進階...

叫出文字編輯器設定的界面。

文字編輯器設定¶

用來調整文字編輯器的設定。

編輯器模式

設定想使用的編輯器，可以分別使用富文字編輯器和 SVG 原始碼編輯器，或是同時使用。

色彩

在這裡你可以設定 SVG 原始碼的語法高亮。

關鍵字

這個功能突顯重要符號如 <，/ 和 >。

元素

突顯元素標籤名稱的格式。text 和 tspan 是元素名稱的例子。

屬性

突顯標籤屬性的格式。如：font-family，當這個不在 style時通常會被寫成一個屬性。

明度

突顯屬性的明度的格式。

注釋

突顯 XML 注釋，格式為：<!-- 這是一個 XML 注釋 -->。注釋是一段不會受任何影響的文字。

編輯器文字色彩

編輯器的主要顏色。

編輯器的背景色彩

編輯器的主要背景色彩。

字型

這個選項可以以輸入系統為基準過濾文字編輯器的字型。有些系統有著大量能正確顯示海外語言的字型，但這些字型未必對你有用。勾選你的系統使用的語言，然後列出支援該語言的字型。

在 SVG 原始碼分頁做出精細的排字¶

由於富文字編輯器無法控制 SVG 文字所容許的所有功能。如要使用那些功能，你需要直接使用 SVG 原始碼。在開始之前，你要先去文字編輯器設定並啟用 SVG 原始碼 或 兩者 的編輯器的模式。富文字編輯器會丟失部分資訊，因此如要保留所有資訊，使用 SVG 原始碼。

Word-spacing, Letter-spacing and Kerning（間距，字距和字距調整）¶

這三個格式都能在富文字編輯器中輸入及顯示，但只有兩個能在富文字編輯器中控制。

字距調整（Kerning）

字距調整在 SVG 版本 1.1中<https://www.w3.org/TR/SVG/text.html#KerningProperty> 表現和 在 CSS 裡的 font-kerning 有些微差異。Krita 預設在這裡使用 auto 屬性，也表示預設是開啟的。如要關閉字距調整，請在文字裡的 style 使用 kerning: 0。任何其他的數值會被加到目前的 letter-spacing。

<text style="kerning:0; font-family:Dancing Script; font-size:18pt; font-size-adjust:0.265625">
    <tspan>No Kerning on Valhalla Tower.</tspan>
</text>
Letter-spacing（字距）

這是字母之間的間距，通常以點為單位。只需在 style 裡加入 letter-spacing然後在後面加上以點為距離的數值。當數值為負時會減少字詞之間的數值。

Word-spacing（間距）

這是用來設定單詞之間的額外距離，預設以點為單位。預設中 word-spacing: 0; 只會用於該字型空白符號的寬度。當數值為負時會減少單詞之間的空間：

<text style="font-family:Noto Serif; font-size:12pt; font-size-adjust:0.389915; text-anchor:middle">
    <tspan>No Adjustment.</tspan>
    <tspan style="letter-spacing:2" x="0" dy="22pt">Letter spacing: 2</tspan>
    <tspan style="letter-spacing:-2" x="0" dy="22pt">Letter spacing: -2</tspan>
    <tspan style="word-spacing:5" x="0" dy="22pt">Word spacing: 5</tspan>
    <tspan style="word-spacing:-5" x="0" dy="22pt">Word spacing: -5</tspan>
</text>

x, y, dx, dy¶

這些是更為精細的控制，能用於定位文字。然而，他們不能可靠地轉換至富文字編輯器格式，因為富文字編輯器使用這些數值一來判斷是否為新一行文字並輸入文字。

X 和 Y

X 和 Y 是絕對座標。但因為你無法由文字編輯器更改絕對座標，這些數值會在 tspan 出現時加入定位中。

dx 和 dy

這些數值是相對座標，對應上一個字母的位置。

字型拉伸和小型大寫字母¶

這些項目也能在富文字分頁的內置格式中儲存及寫入，但不會被用在顯示的文字物件中。

Dominant Baseline, Alignment baseline, Font-size-adjust, Writing mode, Glyph-orientation, rotate（主導基線，對齊基線，字體大小調整，書寫模式，字形方向，旋轉字元）¶

這些屬性目前未能儲存在富文字中，雖然這些屬性能寫進 SVG 文字中，但是不會反映在 SVG 文字形狀中。

Krita 會把富文字中生成的文字加上 font-size-adjust，這樣設計能使設計師方便地使用 SVG 原始碼來進行後期的調整。
