磁性選取區域工具¶

磁性選取區域工具的圖示是選取區域上有一個磁石。這個工具能夠自由地選取區域，但是不像Polygonal Selection Tool或是手繪選取區域工具一般，這個工具會嘗試把選取邊界吸附在高對比的影像上，以簡化選取區域。

這裡有兩個磁性選取區域的方法：

首先用 來設置磁性選取區域的錨點， 直到要完成選取時，點選第一個錨點兩次來完成選取，或是換一下 Enter 來結束，又或是在工具選單中點選 完成 按鈕。

第二個模式，交互模式，在影像中 + 拖曳以選取區域。

你可以透過 拖曳之前節點來編輯選取區域。把錨點拉出畫布外可以把錨點移除。當路徑封閉後，可以按 Shift + Z 來返回上一步。一個未完成的磁性選取可以按 Esc`或是在工具選項按 :guilabel:`取消 按鈕。

重要

磁性選取區域工具大多數動作和其他選取區域工具相似，請詳閱選取區域以取得使用這個工具的更多資訊。

快捷鍵和相黏鍵¶

  • （過時）R 來把選取改變為工具選項裡操作的「取代」，這是預設模式。

  • （過時）A 來設定選取操作為選項中的「相加」。

  • （過時）S來設定選取操作為選項中的「減去」。

  • Shift + 設定選取操作為「相加」。在拖曳時能夠放開 Shift 鍵，但是操作會還是「相加」，其他的操作類同。

  • Alt + 設定選取操作為「減去」。

  • Ctrl + 設定選取操作為「取代」。

  • Shift + Alt + 設定選取操作為「相交」。

在 4.2 版本新加入:

  • 把游標移到選取區域的虛線時，游標會顯示成移動工具圖示，這時候可以 然後拖曳選取區域。

  • 會開啟一個快速選單，裡面會有修改選取區域的選項。

小訣竅

你可以在 Krita 的「一般」設定裡交換 Alt 鍵和 Ctrl 鍵的修飾鍵功能。

小訣竅

這個工具預設並沒有綁定任何快捷鍵，如果想要設定一個，在設定 ‣ 設定 Krita ‣ 鍵盤快捷鍵然後搜尋「手繪選取區域工具」，在這裡可以設定你想要的快捷鍵。在「鍵盤快捷鍵」設定取得更多資訊。

工具選項¶

模式

這個選項在像素及向量選取區域類型中已被解釋。

操作

這個選項在像素及向量選取區域類型中已被解釋。

反鋸齒

這個選項控制是否要羽化邊緣。有些人會想要選取鋸齒狀的邊緣。

過濾半徑：

這個數值控制邊緣判定的範圍。這個數值決定這個工具在對比度判斷的激進程度。數值較低時只有最明顯的對比會判斷為邊緣。數值較高時細微的對比會判斷為邊緣。數值範圍在 2.5 到 100 之間。

閾值：

從 0 到 255，判斷邊緣的銳利度，0 是最不銳利，255 最銳利。只能在交互模式使用。

搜尋半徑：

這個數值決定工具在影像中搜尋銳利邊緣的範圍大小。更大的範圍表示在放錨點時需要更少的精確度，但是這樣的話 Krita 需要運算更久，因而減慢這個工具的速度。

錨點間距：

在使用 拖曳時會自動放置錨點，這個數值會判斷兩個錨點之間的平均間距。數值較低時因錨點較多而精確度更高，但是這樣會導致後續修改困難。數值判斷是螢幕顯示的面積而非影像面積，這代表著這個數值會因縮放影響。

備註

反鋸齒只能像素選取區域模式中使用。
