多邊形選取區域工具¶

這個工具，會有一個帶有虛線的多邊形，能夠以多邊形選取區域。如要建立一個多邊形選取區域， 來放置多邊形的節點，要完成多邊形的話，點選起點或是 兩次來完成多邊形選取區域。

重要

大多數動作和其他選取區域工具相似，請詳閱選取區域以取得使用這個工具的更多資訊。

快捷鍵和修飾鍵¶

  • Shift + 設定選取操作為「相加」。在拖曳時能夠放開 Shift 鍵，但是操作會還是「相加」，其他的操作類同。

  • Alt + 設定選取操作為「減去」。

  • Ctrl + 設定選取操作為「取代」。

  • Shift + Alt + 設定選取操作為「相交」。

在 4.2 版本新加入:

  • 把游標移到選取區域的虛線時，游標會顯示成移動工具圖示，這時候可以 然後拖曳選取區域。

  • 會開啟一個快速選單，裡面會有修改選取區域的選項。

小訣竅

你可以在 Krita 的「一般」設定裡交換 Alt 鍵和 Ctrl 鍵的修飾鍵功能。

小訣竅

這個工具預設並沒有綁定任何快捷鍵，如果想要設定一個，在設定 ‣ 設定 Krita ‣ 鍵盤快捷鍵然後搜尋「手繪選取區域工具」，在這裡可以設定你想要的快捷鍵。在「鍵盤快捷鍵」設定取得更多資訊。

工具選項¶

模式

這個選項在像素及向量選取區域類型已被解釋。

操作

這個選項在像素及向量選取區域類型已被解釋。

反鋸齒

這個選項控制是否要羽化邊緣。有些人會想要選取鋸齒狀的邊緣。

備註

反鋸齒只能像素選取區域模式中使用。
