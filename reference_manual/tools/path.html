貝茲曲線工具¶

你可以透過這個工具來畫出曲線。 來給繪製曲線的起點，然後再點一下來連續控制曲線的錨點。

Krita 會在你新增一個控制錨點時顯示一個有著兩個控制項鐵人的藍色線段，你可以拖曳這些控制項來改變曲線在那個錨點的方向。

在向量圖層中，你可以點選上一個錨點來修改。在中途的錨點（即除了首尾的兩個錨點），你可以分別移動控制項方向來讓曲線的起點和終點的方向不同。在編輯錨點後，點選畫布來繼續為曲線新增錨點。

按 Del 鍵會移除曲線中目前選取的錨點。在曲線上的任何錨點按 兩次或是按 Enter 兩次來完成曲線，或是按 Esc 來取消曲線。在按著 Ctrl 時使用 鍵會把整個曲線移動到不同位置。

在拉線條時按 Ctrl 鍵會推動兩邊的控制柄；而按 Alt`鍵會在曲線中生成銳角；按 :kbd:`Shift 鍵會在曲線末端生成控制柄。 會退回至上一個錨點。

工具選項¶

在 4.1.3 版本新加入:

自動平滑曲線

Toggling this will have nodes initialize with smooth curves instead of angles. Untoggle this if you want to create sharp angles for a node. This will affect curve sharpness from dragging after clicking.

角度吸附增量

曲線吸附的角度。

啟動角度吸附

角度吸附會在下一個曲線為特定角度時十分方便的選項。角度由 角度吸附增量 決定。
