平移工具¶

平移工具可以讓你自由地平移畫布。可以在工具箱底部找到，點選工具後在畫布裡 + 拖曳。

這個工具有兩個快捷鍵，可以快速地從其他工具切換至這個工具：

  • 在畫布中 :kbd:` 空白鍵 +` + 拖曳。

  • 在畫布中 + 拖曳。

若是需要更多有關快捷鍵的資訊，參閱介面導航。
