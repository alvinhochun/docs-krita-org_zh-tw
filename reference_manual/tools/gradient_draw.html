漸變工具¶

在工具面板中能找到漸變工具。在畫布中拉出線條會生成一個順著線條走向的漸層。如果在已經選取了區域的情況下，就會變得像填色工具一樣，上色區域限制在選取區域中。

工具選項¶

形狀：
線性

這個選項會順著方向生成直線的漸變。

雙線性

這個選項會生成一個直線漸變，但是拉出的直線會有鏡像。

放射

這個選項會從下筆點中央向外畫出漸變。

方形

這個選項會從下筆點中央向外畫出方形漸變。

圓錐

這個選項會從下筆點畫出環繞中央的漸變。

對稱圓錐

這個選項會從下筆點畫出環繞中央的漸變，但是會鏡像環繞一次。

螺旋

這個選項會從下筆點畫出螺旋狀漸變。

反向螺旋

這個選項會從下筆點畫出螺旋狀漸變，但是與畫出筆畫方向相反。

按輪廓漸變

這個選項會透過選取區域或圖層的形狀來畫出漸變。

循環：
無

這個選項會把漸變延伸到無限遠。

前鋒

這個選項會往一個方向重覆漸變。

交替

這個選項會重覆漸變，交替畫出正常方向和反轉方向。

反鋸齒閾值

這個選項控制重覆漸變邊界的順滑度。

  • 數值為 0 時代表沒有進行平滑。邊界會有鋸齒。

  • 數值高於 0 時表示邊界裡多少像素是 Krita 需要進行平滑的。

反轉

反轉漸變的方向。

抖色

在 5.0 版本新加入.

8 bits 的色彩深度不足以生成一個真正平滑的漸變。這個選項會透過增加藍色雜訊來在 8 bit 中抖色以減輕漸變的鋸齒。
