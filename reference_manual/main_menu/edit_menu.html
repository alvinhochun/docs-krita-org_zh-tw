「編輯」選單¶

復原¶

取消上一個動作。快捷鍵：Ctrl + Z

重做¶

重做最後取消的動作。快捷鍵：Ctrl + Shift+ Z

剪下¶

  • If shapes are selected, then they are cut from the image and saved into the clipboard.

  • If 選取區域 are active, then that area is cut in all the layers selected in the layer docker and saved into the clipboard.

  • If no selection, the layers selected inside the layer docker are cut from the image and saved into the clipboard.

  • Shortcut: Ctrl + X

複製¶

Similar to Cut, but elements are only copied into the clipboard without modifying existing image. Shortcut: Ctrl + C

剪下（銳利）¶

這個功能可以防止剪下範圍的邊緣出現半透明狀態，讓貼上的內容只有不透明和完全透明。

複製（銳利）¶

與「剪下（銳利）」功能類似，但只做複製而不會剪下原始內容。

合併複製¶

在全部圖層上複製選取區的內容並將其合併。快捷鍵：Ctrl + Shift + C

貼上¶

  • If vector shapes are in the copied buffer, the shapes are pasted into the image as a new layer.

  • If layers are in the copied buffer, these layers are pasted into the image above the active layer or inside a group layer if the active layer is a group layer.

  • Shortcut: Ctrl + V

在游標處貼上¶

與「貼上」功能類似，但這個會對齊游標位置貼上。快捷鍵：Ctrl + Alt + V

貼上到使用中圖層內¶

將剪貼簿之內容貼入當前使用的圖層中。

在 5.0 版本新加入.

貼上為新影像¶

將剪貼簿之內容貼上為新建立的影像檔。快捷鍵：Ctrl + Shift + N

做為參考圖貼上¶

將選擇之內容貼上為新的參考影像。

貼上形狀樣式¶

這功能配合「形狀選取工具」使用，先複製向量形狀所包含之樣式（例如填充、輪廓和特徵），然後以此功能將樣式貼在另一個向量形狀中。

在 4.4.2 版本新加入.

清空¶

Empty the currently selected area or total area of all the selected layers. Shortcut: Del

用前景色填充¶

使用前景色填充圖層或選取區，但不包含混合模式或不透明度。快捷鍵：Shift + Backspace

用背景色填充¶

使用背景色填充圖層或選取區，但不包含混合模式或不透明度。快捷鍵：Backspace

用圖案填充¶

以當前選用的圖案填充圖層或選取區，但不包含混合模式或不透明度。

填充前景色（含不透明度）¶

使用前景色填充圖層或選取區，包含混合模式或不透明度。快捷鍵：Ctrl + Shift + Backspace

填充背景色（含不透明度）¶

使用背景色填充圖層或選取區，包含混合模式或不透明度。快捷鍵：Ctrl + Backspace

填充圖案（含不透明度）¶

以當前選用的圖案填充圖層或選取區，包含混合模式或不透明度。

對選取形狀描邊¶

這個功能會另外建立新圖層並以當前筆刷和它的顏色、粗細…對向量形狀之外框進行描邊。

對選取區域描邊¶

這個功能會打開一個對話盒設定，可對選取區的外框以使用中筆刷在當前圖層進行描邊。
