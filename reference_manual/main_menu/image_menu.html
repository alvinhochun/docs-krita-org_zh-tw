「影像」選單¶

屬性…

顯示影像的屬性內容，包含當前影像的色彩空間、尺寸大小之類資訊。

影像背景色與透明度…

變更當前畫布的背景顏色。

轉換影像色彩空間…

將當前影像轉換至新的色彩空間。

修剪至影像大小

將所有圖層修剪至影像的尺寸大小。這個功能可以降低檔案的大小，但也會移除畫布區域以外的影像資料。

修剪至目前圖層

這是偷懶的方式，此功能可以將整個影像修剪為目前圖層的大小。

修剪至選取區域

這是個偷懶的方式，此功能可以讓 Krita 將當前畫布修剪至選取區域的大小。

旋轉

將影像旋轉至所設定的角度。

切變影像…

對當前影像進行斜切變形，您可以在對話盒中設定切變角度。

水平翻轉影像

以影像中心為軸左右翻轉影像。

垂直翻轉影像

以影像中心為軸上下翻轉影像。

縮放影像至新尺寸…

打開縮放影像尺寸的對話盒，以設定您要縮放的大小。快捷鍵：Ctrl + Alt + I

平移影像…

以您在對話盒中設定的距離，將所有圖層的內容移動到指定之位置。

調整畫布大小…

變更畫布的尺寸，但不要與「縮放影像至新尺寸」混淆。快捷鍵：Ctrl + Alt + C

分割影像

顯示分割影像對話盒。

子波分解…

對當前圖層使用Wavelet Decompose的處理。

分離影像…

將當前影像以Separates依色彩通道拆解成多個圖層。
