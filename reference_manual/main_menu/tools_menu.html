「工具」選單¶

這個選單原有三個項目，但在 Krita 3.0 之後的版本僅保留一個 「指令稿」。

指令稿¶

當安裝與啟用某些 python 指令稿之後，安裝完成的項目會顯示在這個子選單中。

Recording（錄製巨集）¶

在 3.0 版本開始棄用: 由於長久沒有維護又容易造成當機之類的錯誤，因此在 Krita 3.0 版已停用錄製巨集和播放巨集的功能。

這是錄製巨集的功能。可以按下開始接著繪製內容以記錄然後按下停止來完成這個操作。這個功能可紀錄筆劃。產生的資料被儲存為 *.kritarec 檔案。

Macros（巨集）¶

在 3.0 版本開始棄用: 由於長久沒有維護又容易造成當機之類的錯誤，因此在 Krita 3.0 版已停用錄製巨集和播放巨集的功能。

播放或編輯 Krita 錄製的巨集檔案。編輯的功能只能變更筆劃上的筆刷預設或是添加和刪除濾鏡。
