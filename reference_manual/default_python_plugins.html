預設安裝的 Python 外掛程式¶

這個頁面描述了 Krita 預設可用的所有外掛程式（你不需要安裝它們）。

也參考

如果您想查看一系列用戶自定義製作的 Python 外掛程式，可另外下載和安裝，請參閱 使用者製作的 Python 外掛程式。

要了解如何管理外掛程式，請參閱 管理 Python 外掛程式。

如果你想了解更多關於某個外掛程式的訊息，你可以通過進入 設定 ‣ 設定 Krita… 選單，然後選擇 Python 外掛程式管理器 選項卡來訪問該外掛程式的手冊。然後你可以點擊一個特定的外掛程式，手冊將出現在底部的文本區域。

易用性¶

混色滑動條工具面板

提供一個面板，顯示在當前顏色和其他選定顏色之間的多個漸變色條，以便選取一種顏色。

調色板工具面板

提供一個易用的調色板面板，用來添加一般色彩樣本、分組、匯出調色板設定或者調色板本身，支持匯出為 GIMP 調色板文件或者 Inkscape SVG 文件。

快速設定工具面板

提供一個快速設定面板，預先列出了常用的不透明度、流量、筆刷大小。

十大筆刷

這個外掛程式最多可以自定義十個筆刷的快捷鍵。從主選單選擇 工具 ‣ 指令稿 ‣ 十大筆刷 即可打開對話框。頂部顯示了十個方框，每個方框下方顯示了對應的快捷鍵。先從對話框下方的預設列表中選中一個筆刷，然後再點擊上方的方框，即可將選中的筆刷對應到快捷鍵。

若要自定義您的快捷鍵，可在 設定 ‣ 設定 Krita ‣ 鍵盤快捷鍵 對話框中編輯配置，然後更改「十大筆刷」下的「啟動筆刷預設」操作。

改善作業流程¶

漫畫專案管理工具

簡化漫畫創作的外掛程式。

  • 組織頁面並快速訪問頁面。

  • 使用適當的中繼資料匯出多種格式。

  • 編輯中繼資料時提供隨機建議，避免在建立專案之前花時間尋找完美的標題。

批次匯出器

遊戲開發者和平面設計師的外掛程式。

  • 可以將資產批次匯出為多種大小、文件類型和自定義路徑。

  • 使用智慧重新命名工具快速更名圖層。

  • 匯出所有圖層或僅匯出選定圖層。

預設情況下，外掛程式會在 Krita 文件旁邊的export資料夾中匯出影像，並遵循您的圖層堆疊結構。

影像／文件操作¶

指定特性檔案對話框

可以將特性檔案分配給影像，而不是將其轉換為該特性檔案。不同之處在於它只允許通過新特性檔案解釋顏色，而不能更改任何值。它可以在 工具 ‣ 為影像指定色彩特性檔案… 中找到，它將顯示當前影像顏色模型的特性檔案列表。

色彩空間

可選擇文件並將其顏色轉換為新的色彩空間，如 RGBA、CMYKA 或 L*a*b。

色彩通道轉換為圖層

將一個圖層的色彩通道拆分為子圖層。

檔案工具

允許您選擇一個文件並在一個操作中縮放、裁切和旋轉。

濾鏡管理器

對選定的文件快速應用濾鏡效果。

高通濾波

對使用中的文件應用高通濾波濾鏡。

檔案操作¶

匯出圖層

允許您以有序且合理的方式選擇文件並匯出其圖層。

最近檔案

顯示最近打開的文件為縮圖的指令稿。

Python 指令稿¶

Krita 指令稿起步助手

幫助設定 Krita 在執行指令稿時希望看到各種文件的指令稿，亦即：

  • .desktop 中繼資料檔案;

  • 外掛程式主目錄;

  • __init__.py 檔案;

  • 外掛程式的 Python 主檔;

  • Manual.html 使用說明文件;

Python 外掛程式匯入工具

從 zip 檔匯入 Python 外掛程式。請參閱 管理 Python 外掛程式。

指令稿執行器

一個小型 Python 指令稿控制台，允許在編輯器中寫程式並執行它，並提供與執行輸出相關的回饋。您還可以使用「調校」按鈕來除錯程式碼。

十大指令稿

與十大筆刷類似，這個外掛程式最多可以設定十個 Python 指令稿的快捷鍵。
