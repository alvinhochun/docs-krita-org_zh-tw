偏好選項¶

Krita 可讓使用者自訂這個軟體的各項功能。大多數項目都可以在「設定 ‣ 設定 Krita…」對話視窗裡面進行設置。至於 macOS 系統上，這個功能和其它 macOS 軟體一樣顯示在左上角軟體名稱的選單中。

Krita 軟體設定是儲存於一個叫做 kritarc 的檔案中。 Windows 作業系統是放在 %LOCALAPPDATA%\ 目錄下； Linux 是 ~/.config 目錄； macOS 則是 ~/Library/Preferences 目錄。您可以將這個檔案備份到別的地方存放，在不同電腦之間同步或還原時也僅需將此檔案複製到前述的目錄下，值得一提的是此檔案在不同作業系統的 Krita 是通用的！

若您的 Krita 是經由 Microsoft Store（微軟商店）安裝的，那麼這個 kritarc 檔案儲存目錄將會是位於：

%LOCALAPPDATA%\Packages\49800Krita_隨機字串\LocalCache\Local\kritarc

自定義快捷鍵是另外儲存在 kritashortcutsrc 檔案中。它跟 kritarc 一樣都可以自行備份。在本章「鍵盤快捷鍵」中我們會對自定義快捷鍵有更深入的介紹。Krita 設定項目如下（譯者註：受檔案系統限制，以下項目是依照英文原名順序）。
