「鍵盤快捷鍵」設定¶

一套適當的快捷鍵設置可以提高工作效率。您可以自訂 Krita 快捷以符合本身使用習慣，若您由 PhotoShop 或 MyPaint 軟體轉移過來，或覺得自訂的快捷鍵在作業上更有幫助，那麼 Krita 自訂快捷鍵功能可以滿足您的需求。您可在「設定 」‣ 「設定 Krita…」打開設定視窗進入「鍵盤快捷鍵」分頁裡面開始配置您的快捷鍵。

Krita 大部分快捷鍵都可以在「設定」‣ 「Krita 設定...」 ‣ 「鍵盤快捷鍵」頁面中進行設定。此處設定都是簡單的組合鍵，例如 Ctrl+X 快捷鍵可以用以剪切。某些快捷鍵可以按順序組合，例如按 Shift+S 可以叫出吸附選單，然後按 TAB 鍵來選取吸附選項（譯者註：原文中 B 按鍵於此處似乎無效，B 鍵現為手繪筆刷工具的熱鍵）。Krita 還有一個特殊介面用來設定發送到畫布的滑鼠與觸控筆事件，可以在「畫布輸入」設定中找到。

設定方法：只需在「搜尋」欄位中輸入操作，然後給它指定快捷鍵。假設您想把 Ctrl + G 指定給圖層的操作，以便於每次按下 Ctrl + G 後，自動建立一個新圖層群組，請依照以下步驟：

  1. 輸入"快速建立群組"。

  2. 以滑鼠點一下「快速建立群組」，這時會打開一個小對話盒。

  3. 點一下「自訂」單選按鈕。

  4. 接著點一下旁邊的「無」之後會變成「輸入」，請在這裡按下Ctrl + G組合鍵。

  5. 單擊右下方的「確定」即可。

從現在開始，當您按下 Ctrl + G 之後，都會自動建立一個新圖層群組。

小訣竅

靈活使用快捷鍵可以為您節省大量時間，並進一步簡化您的工作流程。

在 5.0 版本新加入: 您可以隨時按下 Ctrl + Enter 組合鍵開啟搜尋操作輸入以搜尋所有可用的功能，從而減少不必要的快捷鍵分配。您也可以為這個功能另外指定快捷鍵。

選單項目¶

搜尋欄位

在這裡輸入文字可以在列表中搜尋符合輸入字串的快捷鍵。

快捷鍵列表

快捷鍵在列表中依照屬性分組，每一個操作都可以有一個主快捷鍵和備選快捷鍵。

載入/儲存快捷鍵設定檔

此設定頁面底部有一排按鈕，可用於建立、刪除、儲存、載入和列印目前快捷鍵方案。

設定¶

主要和備選快捷鍵

每個操作項目都分配了一個預設值，但可能是空的設定。使用者最多可以指派兩個自定快捷鍵，稱為「主要快捷鍵」和「備選快捷鍵」。只需要單擊"自訂"按鈕並按下您想指派給操作項目的快捷鍵組合。若按下的組合鍵已被別的操作佔用，將會提示您目前快捷鍵有衝突 ，讓您選擇要如何處理。

快捷鍵方案

您可以從快捷鍵方案選單中隨時切換不同快捷鍵方案。Krita 附帶了 PhotoShop和 Paint Tool Sai 的預設快捷鍵設定。您可以將額外的快捷鍵方案保存在 ~/.config/krita/input/ 資料夾中。

儲存、載入和分享自訂快捷鍵

您可能希望將快捷鍵的設定沿用在不同電腦、不同平台或與別的使用者共享設定。這可以按下「儲存」/「載入」下拉選單來完成。您也可以在 ~/.config/krita/ 資料夾中將 kritashortcutsrc 手動複製來備份或覆蓋新的快捷鍵方案。此外也可以導出現有方案預設值與當前自訂項目合併以產生新的自訂快捷鍵方案。
