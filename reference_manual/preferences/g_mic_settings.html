「G'Mic」設定¶

G'Mic 的全稱是 GREYC's Magic for Image Computing（影像計算魔術），是個自由開源的濾鏡程式框架。但 Krita 的 G'Mic 只有在 Windows 與 Linux 平台才能使用。

Krita 與 G'Mic 的整合已經歷過很長的時間，現在它更強大也更穩定。Krita 的Windows 版本已經內建了 G'Mic ，安裝好 Krita 之後就會自動找到 G'Mic 外掛程式，而 Linux 平台我們也有製作獨立的 G'Mic 安裝包可供下載。

Linux 的 G'Mic 安裝包設定方式如下：

  1. 下載 gmic-Krita 應用程式。

  2. 將下載後的檔案標記為可執行檔案。

  3. 跳到「設定」 ‣ 「設定 Krita…」‣ 「G'Mic 外掛程式」將檔案路徑指定到 G'Mic 所在位置。

  4. 重新啟動 Krita。

更新 G'Mic 濾鏡¶

G'Mic 外掛程式視窗底部有個「更新濾鏡」的按鈕，您必須連上網際網路才可下載新版的濾鏡來更新。

若是通過外掛程式更新的方式出現問題時，您也可以改以手動進行濾鏡的更新。將更新錯誤出現之對話框中顯示的 URL 複製下來，然後貼到網路瀏覽器的網址列進行下載 ，取得附檔名為 ".gmic" 之檔案，只要將它放到下列目錄即可。

  • Windows 作業系統：%APPDATA%/gmic/update2XX.gmic

  • Linux 作業系統：$HOME/.config/gmic/update2XX.gmic

重新打開 G'Mic 對話視窗，取消勾選「互聯網」然後點一下「更新濾鏡」以更新。

在 5.0 版本開始棄用: G'Mic 外掛程式皆已隨 Krita 主程式一起包裝，不需再另外設定安裝。
