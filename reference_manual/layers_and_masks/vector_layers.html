向量圖層¶

警告

此頁面已過時。請前往 向量圖形 來瀏覽更正確的介紹。

什麼是向量圖層 ?¶

向量圖層，也稱為形狀圖層，是一種僅包含向量 ( 由數學公式所計算出來的圖形，和點陣圖是相對詞 ) 元素的圖層。

這就是向量圖層 Krita 在圖層工作介面中的顯示樣子。

在左側顯示圖層的中的向量內容。 內容的右邊有一個點線構成的頁面圖示，有這個圖示出現代表這是一個向量圖層。 再右邊是圖層名稱。 接下來是圖層是否可編輯的圖示 ( 鎖頭 ) 和繼承Alpha圖示 ( α )。 點擊“眼睛圖示”將切換顯示或影藏。 點擊鎖頭圖示讓其鎖起來表示不能編輯，直到再次點擊將鎖頭圖示打開才能再次編輯。

新增一個向量圖層¶

可用兩種方式創建向量圖層。 使用 “ 添加圖層 ” 按鈕中的額外選項 ( + 號右邊的下拉箭頭 )，選擇 “ 向量圖層 ” 項目，來建立一個新的向量圖層。 還可將矩形或橢圓從 向量圖庫 工作介面中拖曳到繪製圖層上。 不過拖曳的目標圖層如果是向量圖層，則形狀將直接添加到其中，而不會新增加一個向量圖層。

在向量圖層中編輯形狀¶

警告

當向量圖層存發生錯誤，通常會轉換為 72dpi 的圖像顯示，不管實際像素大小如何。 這會使操作形狀變得有點困難，即使向量圖層坐標系使用厘米或英寸這類單位作為基礎，但由於精確輸入是以像素作為計算所以還是會轉換。

基本形狀操作¶

要編輯向量元素的形狀和顏色，需要使用基本形狀選取工具。

選擇此工具後，單擊要操作的向量形狀，就可以看到形狀周圍出現參考線。

在向量圖形上形狀工具共有四種操作方式。

變形 / 移動¶

在工具內部單擊並拖動來移動圖形。 按住 Ctrl 鍵會將移動鎖定在一個軸上。

縮放大小 / 縮放拉神¶

這個操作可以拉伸形狀。選擇一個中間點將可沿該軸拉伸。選擇一個角點將在兩個軸向上進行拉伸。 按住 Shift 鍵可以等比縮放物件 按住 Ctrl 鍵會使縮放操作鏡像就像由物件中心縮放般。

旋轉¶

將滑鼠靠近角落點的外側可以旋轉形狀。按住 Ctrl 鍵可以鎖定每45度轉動一次。

傾斜變形¶

將滑鼠靠近中點的外側可以傾斜形狀。

備註

目前無法僅縮放向量物件一側。 開發人員有意識到這會很有用，並將在人力允許的情況下進行處理。

形狀的節點與線條操作¶

雙擊向量物件以進入編輯形狀的節點或曲線。 單擊並拖動一個節點可以移動。 單擊並沿一條線拖動能在兩點之間造成彎曲度。 拖移時按住 Ctrl 鍵會將移動鎖定在一個軸上。

邊線與填充色¶

除了由點和曲線定義之外，形狀還有兩個定義屬性: 填充 和 描邊. 填充 定義填充形狀對象內部空間的顏色、漸變或圖案。 '描邊' 定義沿形狀邊緣的邊框的顏色、漸變、圖案和粗細。這兩個可以使用**描邊和填充**停靠欄進行編輯。 Dock 有兩種模式。 一種用於描邊，一種用於填充。 您可以通過單擊停靠欄上的實心方塊或黑線來更改模式。 活動模式將通過哪個在另一個之上顯示。

**注意 !! 新版已無該介面**這是填充物件時的介面樣貌。 注意最左邊有紅線白色實心方塊在邊框之上。 這告訴我們該物件沒有內填色，因此該形狀的內部會是透明的。

**注意 !! 新版已無該介面**這是描邊時的介面樣貌。

調整邊線屬性¶

筆畫屬性介面可以編輯向量形狀輪廓外觀。

**注意 !! 新版已無該介面**樣式編輯器可以選擇不同的圖案和線條樣式。 寬度選項可更改向量形狀輪廓的粗細。 cap 選項會更改頭尾的顯示方式。 連接選項可更改摺角的顯示方式。

斜接限制控制對象角的顯示程度。 數字越大，允許越過這些點的角越多。 較低的數字會將筆劃限制為較短且較不尖銳的角。

編輯填充屬性¶

注意 !! 新版已無該介面**所有填充屬性都包含在 **描邊 和 填充 介面中。

大紅色 X 按鈕會將填充設定為無，從而使向量形狀內部的區域變得透明。

右邊是實心方塊。 這會將填充設定為顯示在右邊長按鈕裡面的純色，可以按長按鈕右側的向下箭頭進行選擇。 實心方塊的右側是漸變填充按鈕。 這將設定填充為漸變。 這時按長按鈕旁邊的向下箭頭可以選擇漸變。

在 X 下方是一個顯示圖案的按鈕。 這個內部區域將填充一個圖案。 可以按長按鈕旁邊的向下箭頭來選擇圖案。 另外兩個按鈕用於**填充規則**：填充自重疊路徑的方式。
