再製圖層¶

再製圖層可以顯示另一個圖層的更新結果。不能在這個圖層直接繪畫，但可以應用不同的類型圖或遮罩（例如濾鏡圖層或遮罩）來增添效果。

再製圖層的使用範例¶

例如，在繪製某個魔法角色的圖畫創作，希望在他周圍創作一個可以隨著角色而更新的光暈：

  1. 有一個繪製層 ( 下面簡稱 A 圖層 )，在其中繪製角色

  2. 使用新增再製圖層功能建立一個角色繪製圖層 ( A圖層 ) 的再製圖層 ( 下面簡稱 B 圖層 )

  3. 將濾鏡遮罩 ( HSV/HSL 調整 ) 使用於再製圖層 ( B 圖層 ) 上，讓整個角色剪影外型變為白色 ( 或藍色、綠色 … 等 )

  4. 將濾鏡遮罩 ( 模糊 ) 再次使用於再製圖層 ( B 圖層 ) 上，讓他看起來像是 " 發光 "。

這時候如果繼續繪製和添加細節 ( A圖層 ) ，或是更改擦除時 ( 因為擦除會改變輪廓外型 )，Krita 會自動更新再製出來的圖層 ( B 圖層 ) ，讓 " 發光 " 再次套用這個更改過後的內容。

改變再製圖層的來源內容¶

在「圖層面板」中更改一個或多個再製圖層的來源。首先，請在面板中選擇一個或多個再製圖層（按住 Ctrl 左鍵點擊圖層可以增加選取圖層，按住Shift 左鍵點擊圖層可以連續選取多個圖層）。接著，右鍵點擊任何有被選定的圖層。在彈出的選項中，有一個名為 設定複製來源。點選它。將會再彈出一個對話框，並有一個下拉式選項，其中所有可以被再製圖層當作來源的圖層都會在下拉項目中。如果當下選取的是多個再製圖層，那下拉選單中將會替除這些圖層，不然，就會造成重複的共同來源。

以下是圖層替換來源時的判斷規則：

  1. 任何有選擇的再製圖層都是無效的。

  2. 任何無效的再製圖層以及其父層群組或是其再製圖層也是無效的。

  3. 除上述兩點其它均有效。

如果在下拉菜單中選擇一個圖層後，畫面將即時改變預覽結果。這時點擊 確定 就會進行更改. 反之點擊 取消 會放棄更改。如果這時在對話框外進行其他操作，將會應用目前所選圖層進行更改並自動關閉對話視窗。
