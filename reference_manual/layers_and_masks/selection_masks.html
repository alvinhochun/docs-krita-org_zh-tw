局部選取區域¶

局部選取區域可以記住或說留下編輯圖層上的選取區域。 其工作方式和其他影像編輯軟體中額外的通道很類似。 差異是 Krita 能夠把這些選取紀錄分配到特定圖層，並利用點擊該選取圖層的功能圖示來再次執行選擇。 很簡單點擊圖層工作介面中局部選取區域圖層右邊有個虛線輪廓的圓形圖示就可以了 ( 小技巧 : 也可以按著 ctrl 鍵點擊該圖層 )。

建立局部選取區域，首先要有選取區域，在塗層工作面板中對需要建立的圖層按 並選擇 : 新增 > 局部選取區域。

在塗層工作面板中對局部選取區域按 Alt + 的快速鍵，進入隔離編修模式, 可對選取區域進行變形或是繪製，進而修改局部選取區域( 該模式是灰階模式 )。

一個單圖層可以有多個局部選取區域。 再重申一次。 一個單圖層可以有多個局部選取區域 ( Local Selection Masks 縮寫 LSM)。 這很重要，因為這代表，比方說，您可以對一個影像可以有許多不同的選取區域，並將每個區域留存下來 LSM，然後單點一下即可調出留存區域使用。 如果不使用 LSM ( 或是一個圖層只支援一個遮罩 )，那便不得不建立一層又一層的遮罩。 這樣的做法運算效率是不好的，對 Krita 也是，並且軟體會變慢來提供正確的顯示。所以 LSM 是 Krita 中最重要的功能之一！

全局選取區域¶

可以像使用單一圖層這樣的選擇方式編修全域選擇。 不過，在這之前需要將全域選擇的選項開啟。 這邊，請進入 選取 ‣ 顯示全域選取區域遮罩. 開啟後如果有任何選取區域，全域選取顯示會將局部選取區域顯示在所有圖層的最上層。
