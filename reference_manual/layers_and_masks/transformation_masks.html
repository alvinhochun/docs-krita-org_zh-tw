變形遮罩¶

和使用畫筆來影響遮罩不同，變形遮罩是可以使用（移動、旋轉、剪切、縮放和透視）的圖層，而且不用套用或合併圖層上的效果或是濾鏡遮罩。

就像濾鏡遮罩和透明度遮罩一樣，變形遮罩也可以附加到顏料圖層，一樣是非破壞性可隨時修改的。

在 5.0 版本新加入: 變形遮罩也可以與 動畫曲線 結合使用於 Tween 動畫。

增加變形遮罩¶

  • 首先對現有的顏料圖層 ( 或是群組圖層 ) 增加變形遮罩。

  • 請選擇 變形工具 工具。

  • 在“工具選項”工作介面中選擇任一種變型模式，並且在選取變形遮罩的情況下將它應用到圖層上。

  • 點擊工具選項界面下方「套用」按鈕。

  • 切換變形遮罩的眼睛圖示 ( 可見性 ) 可以查看原本圖像和有套用變形之間的差異。

備註

再製 ( 仿射 ) 變換，如移動、旋轉、剪切、縮放和透視，在原始來源( 可能是群組圖層的再製 )更新後立即更新。 其他變換如 網面、外框 和 液化 會佔用更多的處理效能，為了避免浪費效能，Krita 每三秒更新計算一次。

要編輯變形，請選擇變形遮罩，然後在畫面中使用變形工具。 不管當初您選擇什麼變形模式，變形模式都會和當初使用的一致。 如果這時候切換變形模式，變形將被取消還原。
