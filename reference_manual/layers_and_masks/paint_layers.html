顏料塗層¶

顏料圖層是數位創作或影像處理軟體（如 Krita）中最常見的圖層類型 ( 就是最一般的圖層 )。 如果你曾經在 Photoshop 或是 Gimp，中使用過圖層，就會知道它們的工作方式。 簡單說，顏料塗層，也可稱為像素、光柵堆疊圖層，是一種點陣圖式影像（由許多顏色點組成的影像）。

顏料塗層可以應用許多高階的效果，例如塗抹、塗抹加上扭曲。 這讓它成為最靈活的圖層類型。 但是，顏料塗層在縮放時 ( 尤其是放大 ) 無法得到好的結果（放大時會模糊因為它們是點陣圖），並且已應用上的任何效果無法再次編輯。

為了解決這兩個缺點 ( 放大以及叫果無法修改編輯 )，數位創作者通常會提高的解析度 ( 每英寸像素PPI ) 製作。 對於有大量細節的作品，開到 400 到 600 PPI 的解析度設定並不罕見。 而為解決無法編輯修改效果的問題，最好的方式就是利用濾鏡、透明度和變形 … 等遮罩，這類非破壞性圖層的功能。

只要畫布有足夠的解析度，且不需要再調整之前建立的效果，那麼顏料塗層通常就是最終的圖層類型。 如果單擊圖層泊塢窗中的 新增圖層 圖示 ( 一個大大的 + 號 )，就可以建立一個顏料塗層。 當然，也可以用 新增圖層 旁邊下拉選單來建立其他的圖層類型。

新增一個顏料圖層的快速鍵是 Ins 鍵.
