向量圖形¶

Krita 4.0 對向量工具進行了重大的改寫。 所以這是一個解釋向量工具的頁面：

甚麼是向量圖形？¶

Krita 主要是一種點陣圖形的編輯工具，這表示大多數編輯會更改構成圖像為點陣的像素值。

另一方面，向量圖形使用數學來描述形狀。 因為是使用公式，向量圖可以調整成任意大小( 不會有模糊問題 )。

所以說，向量圖形非常適合用於徽標或是橫幅廣告。 但另一方面，點陣圖形其實比較容易編輯，因此向量往往是商業設計的領域，使用精度更精準。

製作圖形的工具¶

首先可以用製作向量圖層來開始繪製向量圖形（按圖層工具介面左下 + 旁邊的向下箭頭按鈕來增添不同的圖層類型）。 然後除了，手繪、動態和多畫筆工具之外的所有常用繪圖工具均可用於向量圖層繪製形狀。

路徑和折線工具是在向量圖層上最常用的工具，因為其自由繪製可以製作出具有活力的形狀。

另一方面 Ellipse 和 Rectangle 工具可以繪製特定形狀，然後對其進行編輯以製作特定的派形或簡單的圓角矩形。

西方書法和文字工具也製作特殊矢量。 西方書法工具是用來繪製類似壓力鋼筆的筆觸，而文字工具則是用來製作一個可以事後再編輯的文字物件。

注意這些工具都會使用當下的畫筆大小以及當下的前景色和背景色，來定義筆觸粗細及顏色。

最後還有一種製作矢量的方法 Vector Image 工具。它可以添加已在 SVG 文件中定義為**符號**的形狀。 與其他工具不同，這些工具有自己的填充和描邊。

排列圖形¶

向量圖層有自己的堆疊層次順序，就像整個檔案圖層有層次一樣。 所以形狀可以改變上下層次順序。 這可以使用**向量圖形排列**工具介面或 Select Shapes 工具進行修改。

向量圖形排列工具介面還可以對形狀進行分組和取消分組。 也可以精確對齊形狀，例如，將它們對齊到中心，或在所有形狀之間排列出均勻的間距。

編輯圖形¶

向量形狀的編輯是使用 形狀選取工具 和 形狀編輯工具 來完成的。

形狀選取工具 可用於選取向量形狀、並對它們進行群組 ( 使用 ), 、取消群組、使用**布爾邏輯操作**將形狀彼此聯集或差集 ( 使用 )、移動它們的上下關係，或做快速變換 ( 到最上或到最下 )。

填充¶

可以選取形狀並改變前景色來更改形狀的內填色。

也可以用 形狀選取工具 選取向量形狀後進到**工具選項**工具介面中的 填充 標籤來更改內填色。

向量形狀可以用純色、漸層色或圖案來填充。

描邊¶

可以用來為筆觸上色的種類與填充使用的種類是相同。

但，描邊標籤頁的下放有更進一步的選項。 比如，粗細、線條樣式或頭尾記號。

坐標¶

可以用 形狀選取工具 移動形狀，也可以在**工具選項**工作介面中的幾何標籤 ( 第一個標籤 ) 中輸入精準的座標。

編輯節點及特殊參數¶

如果選擇了一個形狀快速的雙擊點兩下，工具就會變為 形狀編輯工具 ，如果是文字的話那就會變成 文字工具 。

在使用 形狀編輯工具 時， 可以針對形狀外圍節點進行一般的路徑移動。 對於特殊形狀，如橢圓和矩形，便可以移動節點調節參數或在 工具選項 工作介面中調節參數。

與其他軟體共同作業¶

Krita 4.0 的重大改變是從 ODG 傳換到 SVG。這表示 SVG 會包覆在 KRA 檔案中， 這也表示 Krita 可以很輕鬆的開啟 SVG 這件事格外重要。因為 SVG 是目前最為流通的向量格式。

Inkscape¶

您可以將向量圖形從 Krita 複製貼到 Inkscape，或反向從 Inkscape 複製貼到 Krita。目前僅支援 SVG 1.1 功能，但網格漸變等較小的功能除外。
